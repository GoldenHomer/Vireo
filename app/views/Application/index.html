Welcome to Vireo 1.8!

This is the initial wiring of the Twitter Bootstrap UI package (v2.0.4).

It wires /public/stylesheets/bootstrap.css into main.html, and copies the following files into the public folder...


Stylesheets

  • /public/stylesheets/bootstrap-responsive.css
  • /public/stylesheets/bootstrap-responsive-min.css
  • /public/stylesheets/bootstrap.css
  • /public/stylesheets/bootstrap.min.css

Javascript

  • /public/javascripts/bootstrap.js
  • /public/javascripts/bootstrap.min.js

Images

  • /public/images/glyphicons-halflings-white.png
  • /public/images/glyphicons-halflings.png


I'm not sure what we'll ultimately use from these, but they are the boiler-plate from Bootstrap, so I've included them to start. Feel free to delete or modify any of this work. I've also included them unconditionally in the main template. I can imagine that we'll want to include or exclude some files depending on what part of the application the user is accessing, but for now, they are global.
