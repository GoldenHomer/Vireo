void org.tdl.vireo.export.impl.ExportServiceImpl.ExportJob.doJob ( ) throws IOException

Run the back ground job.

Iterate through the submissions and stream each one into the a zip archived. The archive will be buffered, and sent directly to the ChunkStream to be transmitted to the browser. This means we never have the complete archive on the server at any one time.

Zip a single file and include it in the archive. This method will use the baseName for the entry name, with the extension of the actual file. This is intended to be used at the top level of the export where we want everything to be "submission_#", but for single file exports it's nice to have the correct extension for the file type.
