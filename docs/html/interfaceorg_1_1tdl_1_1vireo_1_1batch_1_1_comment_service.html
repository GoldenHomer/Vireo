Detailed Description

Batch comment to service. This service handles the background tasks of adding a comment and the option of sending an email on a set of submissions.

Add a comment to all submissions identified by the provided filter.

Parameters
filterThe search filter identifying the records to be changed.
commentThe comment to be added.
subjectThe subject of the comment or email message.
visibilityA boolean to flag if the comment is private or public.
sendEmailA boolean flag to specify w an email should be sent.
ccAdvisorA boolean flag that is only effective when sending an email to specify if the advisor should be CC'ed.
Returns
The metadata for the background job to change submissions. Use this object to keep track of the task's progress.
