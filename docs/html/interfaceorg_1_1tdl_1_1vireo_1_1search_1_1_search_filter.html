Detailed Description

A filter search is a set of parameters to search for a set of Vireo submission. The object is used by the SubmissionRepository to filter the set of all submissions by particular criteria.

Implementers should pick on of the child interfaces instead of implementing this interface directly. The two child interfaces are NamedSearchfilter, and ActiveSearchFilter. The named version is designed to be persisted in the database, while the active search filter is designed to be serialized to a cookie value. Both types may be feed to the filter search to produce search results.
