 JpaNamedSearchFilterImpl (Person creator, String name)
 

Detailed Description

Jpa specific implementation of Vireo's Named Search Filter interface.

Importantly, shortly after Vireo 2.0 this class was modified to remove all external dependencies on other JPA objects. It was proved problematic to preserve the foreign key constraints with because they can not be referenced by a JPA query so there was no good way to handle deleting those elements.

Author
Scott Phillips
