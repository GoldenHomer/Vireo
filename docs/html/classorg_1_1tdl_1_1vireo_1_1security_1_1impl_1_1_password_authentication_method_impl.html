Detailed Description

The most basic authentication method for local password-based accounts. No external sources are evaluated for authentication. The username may either be a netid/username or an email address.

Password based authentication.

First users will be identified by their netid, if that lookup fails then we fail back to see if the username is an email address. Once the person is identified the password is verified. If all that is succesfull then the Person is logged into the SecurityContext.
