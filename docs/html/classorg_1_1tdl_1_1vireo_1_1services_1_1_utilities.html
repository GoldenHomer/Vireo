static

Vireo expects an ORCiD to be in the form "xxxx-xxxx-xxxx-xxxy", where x is numeric and y can be numeric or X. Sometimes you might receive an ORCiD as a URL such as "http://www.orcid.org/xxxx-xxxx-xxxx-xxxy", or without dashes. This function attempts to format a given string into the format Vireo would prefer.

Parameters
unformattedOrcid- An orcid as a string in various formats.
Returns
formattedOrcid - An orcid formatted as just an id with dashes. *
