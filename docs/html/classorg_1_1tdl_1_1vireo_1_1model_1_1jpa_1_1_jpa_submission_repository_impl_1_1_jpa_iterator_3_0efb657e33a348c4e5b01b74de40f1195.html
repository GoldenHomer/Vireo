Detailed Description

Inner class to handle iterating over submissions or action logs. This class solves the problem of needing to loop over all objects which may be too huge to load into memory at one time. The iterator addresses the problem by loading the objects in batches, so only a few objects are in memory at any single time.

Parameters
<T>The type of object to iterate over.
