Detailed Description

Each action log represents a discrete change to a submission. Taken together all action logs list the chronologically all the changes that have occurred. Since action logs by their nature or intended to be merely for auditing and documentation purposes the interface does not provide a way to modify an existing log entry. The only modifiable property is the private flag which indicates weather the log should be visible to students.
