Detailed Description

The error report log keeps track of recent error reports for display with Vireo. When an error is encountered and not able to be handled then it should be reported to the ErrorLog. Later an administrator is able to retrieve information about recent errors from the log. The log will only keep a specific number of recent logs, they may be pushed out of the log at anytime.
