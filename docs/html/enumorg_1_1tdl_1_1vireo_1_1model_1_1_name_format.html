Format the name according to the provided name format using all the name parts provided. All nameparts are optional, however a name with no nameparts is just a blank string.

Parameters
formatHow to format the name.
firstNameThe first name, may be null.
middleNameThe middle name, may be null.
lastNameThe last name, may be null.
birthYearThe birth year, may be null.
Returns
A formatted name.
